Here are some of my projects

  • E-Commerce Book Store

    React.JS, React-Router, Hooks, Heroku

    The app is a Prototype of a book store with multi-platform support. It is fully responsive and functional with a clean simplistic design. I used React.JS as my framework to build upon and used HTML, CSS + BEM, hooks, JavaScript ES6+ for the UI and functionality of the app. The book store is built with multi-pages that was acheived using react router. The pages consist of a home page, explore page and a cart checkout. (GitHub Repository & Live Site Link Below)

  • Corpus

    Made With Figma, UX-UI Design

    Corpus is a concept pharmaceutical app designed to keep track of medicine and being able to pull up intructions on how to take medicine with ability to scan your medicine through the text recognition and be prompted with intructions. Corpus consists of many more functions and was built with figma for prototyping. Within this project you will find 2 case study's presented to you and the wireframing and UI design of the application.

  • Google Gmail Clone

    React.JS, React-Router, Redux, Hooks, Firebase

    The Gmail Clone was developed using React as my frontend framework and with the use of react hooks and javascript I was able to make functional component with the project. The project showcases firebase's real time database and authentication. This project consists of a login and logout function powered by firebase's google authentication, Compose messages and mail within your inbox which is was done through the firebase real time database. Note that this is a just a demo and not an exact full functional clone. The purpose of the following is to showcase Firebase's database functions and how it works within the gmail clone application. (Github Repository, Firebase Project Overview & Live Site in links below for more information.)
